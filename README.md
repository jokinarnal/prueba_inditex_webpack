# prueba_inditex_webpack

Para instalar la app:
    npm install

Para ejecutarla:
    Modo development:
        npm run build
    Modo production:
        npm run prod        

Para servirla:
    serve -s build

Nota: 
    hay que tener instalado el paquete serve. 
    En caso de no tenerlo instalado: 
        npm i serve
